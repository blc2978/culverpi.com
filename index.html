Why This Website?

The reason for making this site was to help me explore programming and web design. The site is hosted on my home raspberry pi that is running Apache Webserver. I'm using a book, HTML and CSS by Jon Duckett, as a reference to help me along the way while I'm getting my feet under me. In addition to helping me learn about HTML, CSS, and JavaScript, this site will serve as a networking resource with my contact information, as well as a creative outlet for my photography.
